Ejemplo de Transición CSS

HTML


<div class="animated-div"></div>

CSS


.animated-div {
  width: 100px;
  height: 100px;
  transition: width 2s;
}

.animated-div:hover {
  width: 300px;
}

  • width es la propiedad clave para la animación (Es la propiedad que va a cambiar)
  • transition especifica que indica que queremos una transición suave (Comienza lento, acelera en el medio y termina lentamente)
  • 2s significa que la transición durará 2 segundos
