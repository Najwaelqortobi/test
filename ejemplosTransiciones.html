Ejemplo de Transiciónes CSS

HTML


<div id = "div1" ></div>

<div id = "div2" ></div>

<div id = "div3" ></div>

<div id = "div4" ></div>

CSS


#div1 {transition-timing-function: linear;}

#div2 {transition-timing-function: ease-in-out;}

#div3 {transition-timing-function: ease-in;}

#div4 {transition-timing-function: ease-out;}

  • linear : Velocidad constante de principio a fin
  • ease-in-out : Combina ease-in y ease-out
  • ease-in : Comienza lentamente y acelera hacia el final
  • ease-out : Comienza rápidamente y desacelera hacia el final
